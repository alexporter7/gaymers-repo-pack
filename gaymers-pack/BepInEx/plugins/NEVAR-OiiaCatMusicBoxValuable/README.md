# OiiaCatMusicBoxvaluable

## Overview  
This mod introduces **OiiaCatMusicBoxvaluable** as a valuable entity in REPO.

## Prerequisites  
- **Zehs-REPOLib**

## Features  
- **Spawns on all maps**  
- Adds a **new** music box that plays **Oiia Cat music**, without replacing the original music box.

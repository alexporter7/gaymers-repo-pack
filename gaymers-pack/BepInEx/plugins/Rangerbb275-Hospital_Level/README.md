# Hospital Level
This mod adds in a new Hospital level to REPO!

## Hospital Contains
- Custom valuables
- Custom ambience breaker
- Custom rooms

## Valuables (File names)
- Big Syringe
- Caduceus
- Cube of Hatred
- Drip Bag
- Heart

- Iron Lung
- Magic Heart
- Pill
- Skeleton Arm
- Skeleton Stand
- Stretcher
- Surgical Light
- Syringe
- Wheelchair
